.. _install:

Install
-------------

To use sku, first install it using pip:

.. code-block:: console

   (.venv) $ pip install git+https://github.com/alexcapstick/SKUtils@main#egg=sku



For example::

   >>> import sku


From here, you can use the functions and classes provided, for example::

   >>> sku.PipelineDD()
